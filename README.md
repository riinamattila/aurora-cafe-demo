# Aurora Café – Demo Website

This is a fictional café website created as a frontend portfolio project.

The purpose of the project is to demonstrate:

- responsive layout
- modern UI styling
- component structure
- simple interactive elements

## Features

- responsive navigation
- hero section
- menu and gallery sections
- booking form (demo only)
- simple chatbot demo

Note:  
This website is a fictional demo project.  
No real booking requests or messages are sent.
